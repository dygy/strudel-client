# Product Overview

Strudel is a live coding environment for creating musical patterns on the web. It's a JavaScript port of TidalCycles, enabling algorithmic music composition and live performance through code.

## Key Features
- **Live Coding**: Real-time pattern creation and modification
- **Web-based**: Runs entirely in the browser
- **Pattern Language**: Mini notation and JavaScript-based pattern composition
- **Audio Engine**: Web Audio API integration with sample playback and synthesis
- **Extensible**: Modular package system for different capabilities

## Target Users
- Live coders and algorithmic composers
- Musicians interested in generative music
- Developers building music applications
- Educators teaching computational music
